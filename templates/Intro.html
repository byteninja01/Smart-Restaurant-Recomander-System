üîß Technologies & Methodologies

  • TF-IDF Vectorization: Transforms restaurant features into numerical vectors capturing characteristic importance
  • Cosine Similarity: Measures restaurant similarity through vector angle calculations (0-100% match scores)
  • Flask Framework: RESTful API design with dynamic routing and template rendering
  • Plotly Visualization: Interactive charts, geographic maps, and statistical distributions
  • Multi-Criteria Filtering: Complex query optimization with AND/OR logic and range filtering
  • Pickle Serialization: Pre-computed similarity matrices for sub-second query responses
